Human Pose Estimation (HPE) Academic Session: 2025/2026 Human Pose Estimation (HPE) is a computer vision task that focuses on identifying and classifring the joints in the human body such as elbows, knees: and shoulders from an image or video. Historically, accurate motion capture required marker-based systems where subjects wore specialized suits with reflective sensors, which provided high accuracy but were prohibitively expensive and invasive (Menache: 2000) The field later evolved to "markerless" systems using depth sensors, such as the Microsoft Kinect, which estimated pose from RGB-Depth data. However, the most recent advancement is the ability to perform Monocular RGB Pose Estimation using Convolutional Neural Networks (CNNs)_ These deep learning models can infer 3D skeletal structures from a single standard 2D camera feed without requiring depth sensors (Menache, 2000). This shift is critical for ZenPose, as it allows the application to run on standard smartphones. Modem approaches, such as MediaPipe Pose: utilize a top-down topology. First, a detector locates the used s Region of Interest (ROI), and subsequently, a tracker predicts 33 skeletal landmarks within that region (Google: 2025). Recent studies indicate that these optimized models can achieve real-time performance (30 FPS) on mobile CPUs with high accuracy for physical exercises (Dill et al_, 2023) 2.2.2.
