# PIXEL 2026 Video Presentation Draft - ZenPose

## Guideline Focus

The provided PIXEL rubric section evaluates **Entrepreneurship Opportunity (10%)**. The excellent band expects the project to be **ready to be extended for pre-commercialization**.

For ZenPose, the video should therefore sound like a product showcase, not only an academic explanation. Emphasize:

- Clear market problem
- Working prototype
- Target users
- Unique value
- Commercial potential
- Next steps toward pilot testing and pre-commercialization

## Recommended Video Length

Target **3 to 4 minutes** unless PIXEL gives a different time limit.

Suggested pacing:

- 0:00-0:20 - Hook and project title
- 0:20-0:50 - Problem and target users
- 0:50-1:25 - Solution overview
- 1:25-2:30 - App demonstration
- 2:30-3:10 - Technology and validation
- 3:10-3:40 - Commercialization opportunity
- 3:40-4:00 - Closing impact

## Shorter 2-Minute Version

Hello, my name is Muhammad Danish Aiman bin Muhammad Nazir, and this is ZenPose, a mobile app for real-time yoga pose evaluation using computer vision.

Home yoga is convenient, but many beginners practise without knowing whether their posture is correct. Most yoga apps only provide videos or static instructions, so users do not receive immediate correction. This can reduce practice quality and may increase the risk of strain.

ZenPose solves this by using the smartphone camera to detect body landmarks and analyse posture in real time. The app compares the user's movement with ideal yoga pose templates, then provides instant visual and voice feedback. If the pose is correct, the user receives positive reinforcement. ZenPose has strong entrepreneurship potential because it provides affordable personal guidance using only a smartphone camera. It can be extended into a freemium wellness app, premium guided programs, or B2B packages for schools, yoga studios, and workplace wellness programs. The same technology can also be adapted for physiotherapy and fitness coaching.

The current prototype is ready to be extended toward pre-commercialization, with the next steps being pilot testing, robustness improvements, UI refinement, and beta release preparation.

In conclusion, ZenPose makes home yoga safer, smarter, more engaging, and more accessible. Thank you.
